## --Title --
-> Netflix Clone

## --Description --
-> This project is a Clone of Netflix Website using Html and Css. this is a practice project to use all the advanced css.

In Html, mainly used tags are\
-> Div\
-> Img\
-> Select\
-> button\
-> input\
-> p\
-> table\
-> ul, li\

In Css, mainly used concepts are\
-> float\
-> positon properties\
-> flex box\
